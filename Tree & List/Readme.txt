This library was created provide a ready made Balanced Binary Search Tree and Doubly Linked List for programmers.

How to use:
1. Include "nodes.h" in your program
2. You can recompile the source or just use the *.obj files
3. Create a variable from the structure "List"
4. Create the Node Functions
5. Set the function pointers in the "List" structure to point to your Node Functions
6. Set the head of your Tree or List from your "List" structure
7. And Bingo!! you're ready to go, call the functions as you need them

For more details check out the example
